VennPool

VennPool is a mobile app that aims the alleviate messy communication associated with organizing large carpools. Even though we focused on deploying our mvp on ios, we used React Native as a framework to allow for future scalability to android os. It was my first time working with React Native, so I decided to create a Development journal to keep track of issues and solutions for reference.
